Osiris

Osiris is “a general purpose, scale-to-zero component for Kubernetes”. As explained in the Zero cost preview environments on Kubernetes with Jenkins X and Osiris blog post I wrote, Osiris is an excellent tool to keep a Jenkins X cluster size under control.

I contributed multiple new features and bug fixes to Osiris.

As the original project has now been archived, I’m maintaining a fork on the Dailymotion OSS organization - as well as how to use it in Jenkins X.

Project link: https://github.com/dailymotion-oss/osiris
